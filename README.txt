HOW TO PUT THIS ON GITHUB PAGES

1. Unzip this folder.
2. Your folder should contain:
   - index.html
   - style.css
   - script.js
   - background.jpg
   - envelope.png
   - star.png
   - paper.png
   - fog.png
   - fog2.png
   - fog3.png
   - fog4.png

3. Open your GitHub repository.
4. Click Add file → Upload files.
5. Upload ALL files from this folder.
6. Scroll down and click Commit changes.

If old files are already there:
- Uploading files with the same names replaces them.
- That is fine.

To turn on the link:
1. Go to Settings.
2. Click Pages in the left sidebar.
3. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
4. Click Save.
5. Remove anything written under Custom domain. You do not need it.
6. Wait 1–3 minutes.
7. Refresh the Pages page.
8. GitHub will show:
   Your site is live at https://YOURUSERNAME.github.io/REPOSITORY-NAME/

That is the link to send.
